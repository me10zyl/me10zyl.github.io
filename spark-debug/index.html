使用 java remote debug 功能， 借助intellij idea 远程调试 WordCount 程序。

Java 中进行远程调试步骤：

  1. 在服务端配置启动参数，激活服务端的远程调试功能。
  2. 本地客户端连接调试服务端。
  3. 添加断点并且启动本地程序。
